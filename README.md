# 🚀 AttaFont Engine (v1.0)

**AttaFont** adalah generator fancy text & font Unicode ultra-cepat berbasis web. Aplikasi ini menghasilkan 2000+ variasi gaya tulisan secara instan tanpa memperlambat performa browser.

![AttaFont Banner](public/favicon.svg)

---

## ✨ Fitur Utama

- 🎨 **2000+ Variasi Font Unicode**: Matematika Bold, Cursive Script, Gothic Fraktur, Small Caps, Double-Struck, Fullwidth, Circled, Zalgo Glitch, dan kombinasi bingkai estetik.
- ⚡ **Client-Side Engine High-Performance**: Pemrosesan client-side murni tanpa mendownload file `.ttf`/`.otf` eksternal.
- 📌 **Sticky Input Studio**: Bar input melayang dengan tombol hapus cepat (`X`), penghitung karakter & kata, serta preset contoh teks instan.
- 🎛️ **Custom Symbol Decorator Builder**: Modul rakit bingkai & simbol khusus secara manual.
- 📋 **One-Tap Instant Copy & Toast**: Menyalin teks secara instan ke clipboard dengan pemberitahuan Toast animasi.
- 🏷️ **Kategori & Filter Pencarian**:
  - `All Fonts` (2000+)
  - `Gamer & Nickname` (Free Fire, MLBB, PUBG)
  - `Aesthetic & Bio` (Instagram / TikTok)
  - `Gothic & Classic` (Fraktur & Medieval)
  - `Glitch & Cyber` (Zalgo & Matrix)
  - `Symbols & Emoji`
  - `Favorites` (Disimpan di `localStorage`)
- 🚀 **Multi-Cloud Ready**: Konfigurasi deployment siap pakai untuk **Netlify** (`netlify.toml`) & **Vercel** (`vercel.json`).

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

---

## 💻 Cara Menjalankan Secara Lokal

```bash
# 1. Clone repository
git clone https://github.com/athenas1337/AttaFont.git

# 2. Masuk ke direktori
cd AttaFont

# 3. Install dependency
npm install

# 4. Jalankan dev server
npm run dev
```

---

## 🌐 Deploy

### Netlify
Otomatis mendeteksi `netlify.toml` dengan build command `npm run build` dan publish directory `dist`.

### Vercel
Otomatis mendeteksi `vercel.json` dengan rewrite rule SPA.
